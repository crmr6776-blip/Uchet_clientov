Политика конфиденциальности мобильного приложения «MiniCRM — учёт заказов для мастеров»
Последнее обновление: 2025 г.

1. Общие положения
Приложение «MiniCRM — учёт заказов для мастеров» (далее — Приложение) используется для ведения учёта заказов, клиентов и оплат мастерами и сервисными центрами. Настоящая Политика описывает, какие данные обрабатываются и как они защищены.

2. Обрабатываемые данные
Приложение хранит только те данные, которые пользователь сам вносит:

имя клиента и номер телефона;
описание заказа и текстовые заметки;
стоимость работы, материалов, наценки, оплату и долг;
статус заказа и дату создания/завершения;
голосовые заметки по заказу (аудиофайлы);
фото и видео, снятые из Приложения;
технический идентификатор устройства (Device ID) и срок действия лицензии.
Приложение не собирает геолокацию, аккаунты, историю звонков и другие подобные данные.

3. Хранение и обработка данных
Все данные хранятся локально на устройстве пользователя во внутреннем хранилище Приложения и связанных с ним каталогах.

Разработчик не получает эти данные автоматически и не имеет удалённого доступа к базе заказов.

Передача данных возможна только по выбору самого пользователя, например при:

отправке CSV-отчётов в почту, мессенджеры или другие приложения;
отправке сообщений разработчику через WhatsApp, Telegram и другие сервисы.
4. Доступ к микрофону и камере
Приложение запрашивает доступ:

к микрофону — для записи голосовых заметок по заказу;
к камере — для съёмки фото и видео по заказу.
Полученные медиафайлы сохраняются локально и не передаются разработчику автоматически.

5. Права пользователя
Пользователь может:

просматривать, изменять и удалять заказы и данные клиентов;
очищать базу через раздел Admin (удаление всех данных, восстановление демо-данных);
удалить Приложение с устройства, что обычно приводит к удалению локально хранящихся данных.
6. Ответственность и безопасность
Доступ к данным зависит от настроек безопасности самого устройства (PIN, пароль, отпечаток, Face ID и т.п.).

Разработчик не несёт ответственности за доступ третьих лиц к данным при утере устройства или его передаче другим людям, а также за передачу данных пользователем через сторонние сервисы (почта, мессенджеры и т.п.).
